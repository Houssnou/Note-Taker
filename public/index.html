• Note Taker (current)

Best note taker ever by Houssnou M. KABORE!

This is a Node, Express, and MySQL powered note-taking application.

It uses Express to handle routing and server HTML as well as JSON, MySQL to store notes, and jQuery for AJAX requests and handling other front-end logic.


Additional technologies:
  • Bootstrap Accordions
    • to list all notes and to view the note content just click on the title
  • Bootstrap Modals
    • click on update or delete icon and a modal will pop up
  • Moment NPM to handle date formatted YYYY/MM/DD HH:mm:ss
